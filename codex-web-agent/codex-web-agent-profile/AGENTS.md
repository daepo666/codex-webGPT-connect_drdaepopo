# AGENTS.md

이 프로젝트는 Codex CLI를 LLM 판단기로 쓰고 Playwright로 브라우저를 조작하는 미니 웹 에이전트 예제다.

규칙:
- ChatGPT/OpenAI 웹페이지 자동 수집이나 응답 추출을 목적으로 사용하지 않는다.
- 로그인 우회, CAPTCHA 우회, 보안장치 우회를 하지 않는다.
- 사용자가 직접 로그인한 전용 프로필은 사용할 수 있지만, 비밀번호/2FA/보안 질문은 사람이 직접 처리한다.
- 결제, 계정 삭제, 비밀번호 변경, 개인정보 공개, 대량 발송 같은 민감 행동은 수행하지 않는다.
- action_schema.json 형식의 JSON action만 실행한다.
- 일반 웹 자동화 테스트와 사용자가 허가한 계정/사이트 작업에만 사용한다.
